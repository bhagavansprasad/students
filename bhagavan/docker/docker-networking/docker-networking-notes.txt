- Default Intra-host network communication
1. Create two containers on the same hosts (same docker engine), by default containers can communicate each other
$ docker run --name busybox1 -p 8080:8080 -it busybox
$ docker run --name busybox2 --link busybox1:busybox1 -it busybox

NOTE: in busybox2 you can ping busybox1 by name 
# ping busybox1
